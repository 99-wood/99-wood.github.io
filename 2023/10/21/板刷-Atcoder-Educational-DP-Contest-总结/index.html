前言

最近为了水 ACM 选修课积分, 于是写了一下 OJ 中的 DP 提高题, 发现是 AtCoder 上的题目. 做下来收获颇丰, 所以写篇总结.

总结会写一些我在写的时候花时间比较多或者看了题解或者有收获的题, 代码选择性地放.

J Sushi

题目链接

难在优化状态, 发现答案与盘子的顺序无关, 设计状态 fi,j,kf_{i, j, k}fi,j,k​ 表示有 iii 个盘子有 111 个寿司, 有 jjj 个盘子有 222 个寿司, 有 kkk 个盘子有 333 个寿司全部吃光的期望次数.

状态转移方程推出来是

fi,j,k=ii+j+k×fi−1,j,k+ji+j+k×fi+1,j−1,k+ki+j+k×fi,j+1,k−1+ni+j+kf_{i, j, k} = \frac{i}{i + j + k} \times f_{i - 1, j, k} + \frac{j}{i + j + k} \times f_{i + 1, j - 1, k} + \frac{k}{i + j + k} \times f_{i, j + 1, k - 1} + \frac{n}{i + j + k} fi,j,k​=i+j+ki​×fi−1,j,k​+i+j+kj​×fi+1,j−1,k​+i+j+kk​×fi,j+1,k−1​+i+j+kn​

一个易错点是循环顺序是 kkk, jjj, iii. 读者可以思考一下原因.

R Walk

题目链接

设计状态 fi,u,vf_{i, u, v}fi,u,v​ 表示长度为 iii 的从 uuu 到 vvv 的路径个数.

转移方程

fi,u,v=∑k=1nfi−1,u,k+f1,k,vf_{i, u, v} = \sum\limits_{k = 1}^n f_{i - 1, u, k} + f_{1, k, v} fi,u,v​=k=1∑n​fi−1,u,k​+f1,k,v​

W Intervals

题目链接

好题.

按区间右端点排序, 只有考虑到 i 时, 计算所有右端点为 i 的区间对答案的贡献, 可以做到不重不漏. 而且可以发现, 这样计算只需要考虑最右边选的点的位置.

令 fi,jf_{i,j}fi,j​ 表示考虑到 i, 最后一个点在 j 的最大值.

写出状态转移方程发现可以把 fff 放在线段树上. 很好的一道题.
